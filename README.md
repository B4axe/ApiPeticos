## Descrição

Esta API foi desenvolvida em **Spring Boot** e é parte essencial do ecossistema do aplicativo **Peticos**. Ela realiza operações CRUD (Criar, Ler, Atualizar, Excluir) em diversas tabelas do banco de dados, sendo as mais importantes:

- **`user_`**: Responsável por armazenar e gerenciar as informações dos usuários da plataforma.
- **`pet_register`**: Tabela que armazena os dados dos pets registrados, incluindo informações como nome, idade, raça e características adicionais.

A API oferece suporte para gerenciar perfis de usuários e registros de pets, além de integrar diretamente com o front-end do aplicativo **Peticos**, fornecendo uma camada robusta de backend para garantir o correto funcionamento da aplicação.
